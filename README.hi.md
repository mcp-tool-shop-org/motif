एक ऐसा सॉफ्टवेयर स्टूडियो जो इंटरैक्टिव गेम के संगीत को बनाने, व्यवस्थित करने, रिकॉर्ड करने और निर्यात करने के लिए उपयोग किया जाता है, और जो उपयोगकर्ता की आवश्यकताओं के अनुसार अनुकूलित किया जा सकता है।

## यह क्या है।

मोटिफ एक ऐसा वर्कस्टेशन है जो रचना को प्राथमिकता देता है और अनुकूलन क्षमता को ध्यान में रखता है। यह संरचित संगीत रचना—क्लिप, क्यू, सीन, लेयर, ऑटोमेशन—को एक ऐसे अनुकूलनीय लॉजिक के साथ जोड़ता है जो रनटाइम पर गेम की स्थिति के अनुसार प्रतिक्रिया करता है। इसका परिणाम: ऐसा गेम संगीत जो कृत्रिम नहीं, बल्कि जानबूझकर बनाया गया लगता है।

## यह क्या नहीं है।

एक डिजिटल ऑडियो वर्कस्टेशन (DAW)। एक खिलौना सीक्वेंसर। एक कृत्रिम बुद्धिमत्ता (AI) आधारित संगीत जनरेटर। एक ऐसा डेटाबेस जो ध्वनि के साथ दुनिया बनाने में मदद करता है। "मोटिफ" एक गंभीर रचनात्मक उपकरण है, जिसका उपयोग अनुकूलनीय गेम स्कोर बनाने के लिए किया जाता है।

## यह क्या कर सकता है।

- **रचना (Compose)** — नोट्स, वाद्य यंत्र, स्केल, कॉर्ड, मोटिफ परिवर्तन, तीव्रता भिन्नताएं वाले क्लिप।
- **संश्लेषण (Synthesize)** — मल्टी-ऑसिलेटर सिंथ आवाजें, यूनिसन/सुपरसॉ (16 प्रीसेट), एलएफओ मॉड्यूलेशन (फ़िल्टर, एम्प्लिट्यूड, पिच)।
- **सैंपल वाद्य यंत्र (Sample instruments)** — पियानो, स्ट्रिंग, गिटार टेम्पलेट, सैंपलवॉइस के माध्यम से; इम्पोर्ट, ट्रिम, स्लाइस, किट बिल्डर।
- **व्यवस्था (Arrange)** — लेयर्ड स्टेम, सेक्शन भूमिकाएं, तीव्रता वक्र वाले सीन; 10 ड्रम पैटर्न प्रीसेट।
- **मिक्स और प्रभाव (Mix and effect)** — 8 प्रभाव प्रकार (ईक्यू, डिले, रिवर्ब, कंप्रेसर, कोरस, डिस्टॉर्शन, फेज़र, लिमिटर); प्रत्येक स्टेम के लिए 4 इनसर्ट एफएक्स स्लॉट।
- **एक दुनिया का निर्माण (Score a world)** — मोटिफ परिवार, स्कोर प्रोफाइल, क्यू परिवार, वर्ल्ड मैप प्रविष्टियां, व्युत्पत्ति।
- **स्वचालन (Automate)** — लेन, मैक्रो, एनवेलप, लाइव कैप्चर और मर्ज।
- **पुनर्प्राप्ति और पुन: उपयोग (Recall and reuse)** — टेम्पलेट, स्नैपशॉट, शाखाएं, पसंदीदा, संग्रह, तुलना।
- **एमआईडीआई (MIDI)** — स्टैंडर्ड एमआईडीआई फ़ाइलों का इम्पोर्ट/एक्सपोर्ट।
- **अनुकूलित तर्क (Adaptive logic)** — ट्रिगर बाइंडिंग, ट्रांज़िशन, नियतात्मक सीन रिज़ॉल्यूशन।
- **प्रदर्शन (Perform)** — रीयल-टाइम क्लिप पूर्वावलोकन, क्लिक-टू-ऑडिशन, ऑडियोकॉन्टेक्स्ट-शेड्यूल्ड क्लिक के साथ मेट्रोनोम।
- **सत्यापन (Validate)** — स्कीमा सत्यापन, अखंडता ऑडिट, क्रॉस-रेफरेंस जांच।
- **निर्यात (Export)** — 24/32-बिट WAV, 44.1/48/96kHz पर; गेम इंजन के लिए रनटाइम पैक।
- **लेखन (Author)** — अनडू/रीडू (50 स्तर तक, Ctrl+Z), ऑटोसेव के साथ प्रोजेक्ट सेव/लोड, कीबोर्ड शॉर्टकट (स्पेस=प्ले, ?=मदद), वैश्विक बीपीएम और टाइम सिग्नेचर।
- **विश्वसनीयता (Reliability)** — त्रुटि सीमा, सुचारू रिकवरी, सैंपल-सटीक टाइमिंग के लिए ऑडियोकॉन्टेक्स्ट लुकअहेड शेड्यूलिंग।

## स्थापित करें।

```bash
npm install @motif/schema @motif/clip-engine @motif/runtime-pack
```

सभी पैकेज npm पर `@motif` नाम के दायरे में प्रकाशित किए जाते हैं।

## त्वरित शुरुआत (मोनोरेपो)।

```bash
pnpm install
pnpm build
pnpm test       # 1,116 tests across all packages
pnpm dev        # Start Studio dev server
```

**आवश्यकताएं:** Node.js >= 22, pnpm >= 10।

## परीक्षण।

सभी 16 पैकेजों में यूनिट टेस्ट शामिल हैं जो स्कीमा सत्यापन, डेटा अखंडता जांच, नमूना संचालन, वर्ल्ड स्कोरिंग, ऑटोमेशन, लाइब्रेरी प्रबंधन, प्लेबैक, संश्लेषण, प्रभाव, MIDI और स्टूडियो एकीकरण को कवर करते हैं। सभी पैकेजों में कुल 1,116 परीक्षण हैं।

सब कुछ चलाने के लिए: `pnpm test`

## सुरक्षा और विश्वसनीयता

मोटिफ पूरी तरह से **ब्राउज़र में** चलता है। कोई सर्वर नहीं, कोई क्लाउड सिंक नहीं, कोई टेलीमेट्री नहीं।

- **डेटा जिस पर काम किया जाता है:** उपयोगकर्ता द्वारा बनाए गए साउंडट्रैक पैकेज फ़ाइलें (JSON), ऑडियो एसेट संदर्भ, ब्राउज़र लोकल स्टोरेज।
- **डेटा जिस पर काम नहीं किया जाता है:** कोई सर्वर-साइड स्टोरेज नहीं, ब्राउज़र सैंडबॉक्स से परे कोई फ़ाइल सिस्टम एक्सेस नहीं।
- **नेटवर्क:** शून्य नेटवर्क आउटगोइंग - सभी रचना और प्लेबैक क्लाइंट-साइड पर होता है।
- **गुप्त जानकारी:** यह कोई क्रेडेंशियल नहीं पढ़ता, संग्रहीत नहीं करता या प्रसारित नहीं करता।
- **टेलीमेट्री:** कोई भी डेटा एकत्र या भेजा नहीं जाता।
- **अनुमतियाँ:** केवल मानक ब्राउज़र एपीआई (वेब ऑडियो एपीआई) का उपयोग किया जाता है।

भेद्यता रिपोर्टिंग के लिए [SECURITY.md](SECURITY.md) देखें।
