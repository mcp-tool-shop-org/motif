# @motif/docs

Sitio de documentación de Motif, creado con Astro.

## Páginas

- **Inicio** — descripción general y navegación.
- **Primeros pasos** — configuración y pasos iniciales.
- **Conceptos básicos** — entidades, esquemas y modelo de datos.
- **Flujo de trabajo** — flujos de trabajo y patrones creativos.
- **Ejemplos** — paquetes de ejemplo y uso.

## Desarrollo

```bash
pnpm --filter @motif/docs dev    # Start dev server
pnpm --filter @motif/docs build  # Build static site
```
