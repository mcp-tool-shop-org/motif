# @motif/docs

Sito di documentazione per Motif, realizzato con Astro.

## Pagine

- **Home** — panoramica e navigazione
- **Inizio** — configurazione e primi passi
- **Concetti fondamentali** — entità, schemi e modello dati
- **Flusso di lavoro** — flussi di lavoro e modelli creativi
- **Esempi** — pacchetti di esempio e utilizzo

## Sviluppo

```bash
pnpm --filter @motif/docs dev    # Start dev server
pnpm --filter @motif/docs build  # Build static site
```
