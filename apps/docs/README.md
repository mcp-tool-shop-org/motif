# @motif/docs

Documentation site for Motif, built with Astro.

## Pages

- **Home** — overview and navigation
- **Getting Started** — setup and first steps
- **Core Concepts** — entities, schemas, and data model
- **Workflow** — authoring workflows and creative patterns
- **Examples** — example packs and usage

## Development

```bash
pnpm --filter @motif/docs dev    # Start dev server
pnpm --filter @motif/docs build  # Build static site
```
